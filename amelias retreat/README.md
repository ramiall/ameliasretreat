# Amelia's Retreat - Beauty Massage Spa Website

A professional, elegant website for Amelia's Retreat, a luxury beauty massage spa studio.

## Features

- **Responsive Design**: Fully responsive layout that works beautifully on all devices
- **Modern UI**: Clean, classy design with elegant typography and soothing color palette
- **Smooth Animations**: Subtle scroll animations and transitions for a premium feel
- **Easy Navigation**: Intuitive navigation with smooth scrolling
- **Nature Imagery**: Beautiful nature and wood-themed images for services
- **Contact Information**: Clean contact section with location, hours, and contact details
- **Professional Sections**: Hero, Services, About, and Contact sections

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (Playfair Display & Lato)

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No build process required - it's a static site ready to deploy!

## Deployment to Netlify

This website is ready to deploy to Netlify. You can deploy it in several ways:

### Option 1: Deploy via Netlify UI
1. Push this code to a GitHub repository
2. Go to [Netlify](https://www.netlify.com)
3. Click "New site from Git"
4. Connect your GitHub account and select this repository
5. Click "Deploy site"

### Option 2: Deploy via Netlify CLI
```bash
# Install Netlify CLI
npm install -g netlify-cli

# Deploy
netlify deploy --prod
```

### Option 3: Drag and Drop
1. Zip the project folder
2. Go to [Netlify Drop](https://app.netlify.com/drop)
3. Drag and drop the zip file

## Adding Images

Before deploying, you need to add nature and wood-themed images to the `images/` folder:

1. **massage-nature.jpg** - Nature scene for Signature Massage
2. **facial-nature.jpg** - Wood/nature scene for Facial Treatments
3. **beauty-nature.jpg** - Natural beauty scene for Beauty Services
4. **wellness-forest.jpg** - Forest scene for Wellness Packages
5. **aromatherapy-woods.jpg** - Wood/tree scene for Aromatherapy
6. **couples-nature.jpg** - Nature retreat scene for Couples Retreat

See `images/README.md` for detailed recommendations on where to find suitable images.

## Customization

### Update Contact Information
Edit the contact section in `index.html` to update:
- Address/Location details
- Phone number
- Email
- Business hours

### Update Services
Modify the services in the services section of `index.html` to match your actual offerings.

### Colors
Customize the color scheme in `styles.css` by modifying the CSS variables in the `:root` selector.
